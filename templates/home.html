Welcome to Loan Predictor API

A Flask-based API for loan prediction and validation.

Getting Started

Use the API endpoints to predict and validate loan applications:

  • POST /predict - Submit loan application data and get prediction
  • POST /validate-loan - Validate loan application data
  • GET /health - Check API health status
  • GET /request-info - View request metadata

Example Request

POST /predict
Content-Type: application/json

{
  "age": 35,
  "income": 50000,
  "loan_amount": 20000
}
